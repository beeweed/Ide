## ğŸ¯ Pro Tips

### Save Often
- Get in the habit of pressing `Ctrl/Cmd + S` frequently
- Use `Ctrl/Cmd + Alt + S` to save all modified files at once
- Watch for the blue dot indicator on tabs (shows unsaved changes)

### Quick Navigation
- Use `Ctrl/Cmd + G` to jump to specific line numbers
- Use `Ctrl/Cmd + P` for fast file switching (faster than clicking)
- Use `Ctrl/Cmd + F` to find within current file

### Code Organization
- Create folders to organize related files
- Use descriptive file names for easy searching
- Keep related files in the same folder

### Theme Switching
- Click the theme toggle in status bar to switch between light/dark
- Dark theme is easier on eyes for long coding sessions
- Light theme is better for presentations
